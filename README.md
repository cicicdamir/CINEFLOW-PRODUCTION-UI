## üíé Engineering Excellence (Senior Level)

* **Modular Architecture:** Implements the **Module Pattern** for state encapsulation, preventing global scope pollution and ensuring scalable, maintainable code.
* **Performance First:** Zero React/Vue dependencies. Focused on ultra-fast TTI (Time to Interactive) and silky-smooth 60fps animations.
* **Data Security:** Custom-built data sanitization engine (XSS protection) that neutralizes potential threats from external API responses.
* **Unbreakable Mobile UX:** Advanced CSS logic for bounce-free scrolling control and "snap-to-grid" layouts optimized for iOS and Android browsers.

* **API:** Real-time integration with the **TVMaze REST API**

## üåü Key Functionalities

1.  **Smart Search:** Integrated *debounce* logic that optimizes the number of API calls, preserving server resources while providing real-time results.
2.  **Dynamic Hero Engine:** Fluid background image transitions with automatic metadata synchronization and depth effects.
3.  **Local Persistence:** A fully functional "My List" system utilizing `localStorage` for persistent user preferences across sessions.
4.  **Skeleton Loading:** Custom *shimmer* effects ensuring visual continuity and reducing perceived latency during data fetching.
